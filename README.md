# Editter
### Aplicación para trabajar con pdfs.

## Funciones:
- Visualización de pdf
- Conversor de pdf a word y viceversa
- Cortar pdf
- Unir pdf
- Proteger pdf
- Funciones IA: resumir pdf, preguntar a la IA...
